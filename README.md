# 🧮 Простой Консольный калькулятор (Simple Console Calculator)

Простой консольный калькулятор на C# для выполнения базовых арифметических операций с сохранением истории вычислений в текстовый файл.

## 🚀 Возможности

- Поддержка четырёх арифметических операций: сложение (`+`), вычитание (`-`), умножение (`*`), деление (`/`)
- Ввод двух чисел и выбор действия через консоль
- Отображение результата на экране
- Автоматическое сохранение каждой операции в текстовый файл (`Data.txt`)
- Запись в формате: `число1 оператор число2 = результат`

## 🛠️ Технологии

- C# (.NET)
- Работа с консолью (`System.Console`)
- Преобразование типов (`int.Parse`)
- Запись в файл (`System.IO.File`)
